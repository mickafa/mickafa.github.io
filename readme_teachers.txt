Для работы программы нужны три входных файла:

"Учителя.csv" в кодировке UTF-8
"Учителя1.csv" в кодировке ANSI
"days.txt" в кодировке UTF-8 (в нём перечислены названия дней недели по одному в строчке).

Оба csv файла идентичны друг другу, получаются из самой исходной excel таблицы, из которой нужно удалить ведущие строки и столбцы. Разделители
между клетками - знаки ';' (точка с запятой). Учителя.csv и Учителя1.csv отличаются только кодировкой UTF-8 - ANSI.

Рассчитано на то, что таблица имеет по 7 уроков каждый день, то есть 35 уроков в каждой строке. Соответственно, каждая строка имеет
ровно 36 записей: фамилию учителя и 35 уроков (среди них есть пустые - то есть окна. В программе это учтено).

Количество строк (то есть количество учителей в расписании) не фиксировано, может быть любым, определяется автоматически по ходу выполнения
программы.

В результате работы программы на экран выводится количество обработанных строк (т.е.учителей в расписании), создаются окончательно 
готовые файлы html для каждого учителя в кодировке UTF-8, а также одна сводная таблица-список rt.html в кодировке ANSI. Этот единственный
 файл нужно перекодировать в текстовом редакторе в UFT-8, и готово. Комплект из "количества учителей"+1 файлов можно загружать на сервер.

Сводная таблица - список учителей rt.html формирется в том порядке, в котором учителя находятся в исходном файле эксель. Поэтому
изначально есть смысл сделать сортировку по алфавиту, (или в том порядке, в котором нужно).
